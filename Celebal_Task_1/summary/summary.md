# Assignment 1 - Basic Data Cleaning

## Dataset Used
Combined_dataset.csv

## Tasks Performed

1. Loaded dataset using Pandas.
2. Explored dataset using:
   - head()
   - shape
   - columns
   - dtypes
   - info()
3. Identified missing values.
4. Filled missing numerical values using median.
5. Filled missing categorical values using "Not Available".
6. Checked and removed duplicate records.
7. Selected specific columns.
8. Filtered rows based on rating.
9. Created derived column:

   total_amount = initial_price × ratings_count

10. Exported cleaned dataset as CSV.

- Basic_Data_Cleaning.ipynb
- cleaned_fashion_products.csv
